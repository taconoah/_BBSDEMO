## 极客论坛项目1.0

## 前言

非常简单的Vue3+Node全栈项目,整个项目还有未完善的地方。希望大佬多提宝贵意见。

## 技术栈

#### 前端技术栈

-  **HTML、CSS、SCSS、JavaScript、ES6、Axios、Vue3、Pinia、element-plus**

#### 后端技术栈

- **Node.js、Express.js框架、Mysql**

## 项目运行

- 文件解压

- 进入项目文件夹

- 前端

  1. npm run dev启动项目

- 后端

  1. 更改数据库连接配置
  2. SQL语句导入本地数据库当中
  3. node app.js
